== Description ==

Creates a widget showing the most recent essay data from HeroPress.  Optionally renders featured image, title, contributor, and date.

== Installation ==

1. Upload the `/heropress-recent-essay-widget/` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit Appearance -> Widgets in the admin and place the widget in a sidebar

== Screenshots ==

1. Admin area for the widget area.

1. 2 most recent essaays showing in a widget

== Usage ==

Use the normal widget admin UI to drag the widget into your sidebar.

The data fetcher caches the data for 12 hours.
